## Ferramentas utilizada;

- React Native
- Typescript
- Expo
- Renimated v2
- Moti
- React Native SVG

## Estudo e aprendizado;

Aplicação modelo para estudarmos e aprender um pouco sobre animação;

Clone este projeto e comece com a Expo;
```bash
  git clone https://github.com/SilvaneiMartins/flashcards
  cd flashcards
  npm install or yarn
  expo start
```
